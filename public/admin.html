Firebase Config

Firebase projenizin ayarlarından aşağıdaki bilgileri girin.
console.firebase.google.com → Proje Ayarları → Genel → Web Uygulaması

Henüz Firebase projeniz yoksa:

  1. Firebase Console'da yeni proje oluşturun
  2. Firestore Database'i etkinleştirin (Test Mode)
  3. Authentication → Email/Password etkinleştirin
  4. Users sekmesinden admin hesabı ekleyin
